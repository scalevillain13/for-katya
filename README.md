## Как добавить фото

1. Положи 10 фотографий в папку `images/` с **точными** именами:

| Файл | Что положить |
|------|--------------|
| `photo-1.jpg` | Совместное — первая встреча (25.08.2025) |
| `photo-2.jpg` | Совместное |
| `photo-3.jpg` | Совместное |
| `photo-4.jpg` | Совместное |
| `photo-5.jpg` | Совместное |
| `photo-6.jpg` | Она одна — смешная |
| `photo-7.jpg` | Она одна — смешная |
| `photo-8.jpg` | Она одна — официальная |
| `photo-9.jpg` | Она одна — официальная |
| `photo-10.jpg` | Любимое на выбор |

2. Формат: `.jpg` или `.jpeg` (если `.jpeg` — переименуй в `.jpg` или измени имя в `content.js`).

3. После добавления фото — закоммить и запушить:

```bash
git add images/
git commit -m "Add photos"
git push
```

Сайт обновится автоматически через 1–2 минуты.

## Как редактировать тексты

Все тексты — в файле **`content.js`**. Открой его в любом редакторе и меняй:

- `hero` — заголовок и подзаголовок
- `photos` — подписи и даты под фото
- `reasons` — блок «Почему я скучаю»
- `loveNotes` — короткие послания
- `closing` — финальное письмо и скрытое сообщение
- `relationshipStart` — дата начала отношений для счётчика (формат `YYYY-MM-DD`)

После правок:

```bash
git add content.js
git commit -m "Update texts"
git push
```

## Локальный просмотр

Открой `index.html` в браузере или запусти простой сервер:

```bash
npx serve .
```
